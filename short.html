Windows Shortcut keys

Windows short keys that you need to know.


Computers are considered a necessity even in modern times.
If you are learning a computer or want to know about it, then you must know its shortcut keys, how does it work and how many is the computer shortcut key?
The work is usually done with the help of a mouse on a computer. But you can work more easily with the help of a keyboard on the computer.

Computers have many things to learn. Today, we are going to discuss about computers shortcut key’s in this article?



Q1. What is the shortcut keys?


All times, we have to rely on the mouse to do small tasks on the computer. Through computer shortcut keys, we can also operate the computer through the keyboard. There are some keys in keyboard which make you more comfortable using a keyboard than the mouse.


Q 2.Why do you need to know about shortcut key?


Now you must be wondering why computer key is so important?Then we will try to explain your answer to you by the following points.

1. Computer shortcut keys are used to save time.
2. Using the shortcut keys is considered better than the mouse.
