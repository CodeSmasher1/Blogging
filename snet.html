Computer Network Architecture


Computer Network Architecture is the physical and logical connection between nodes and other devices. It shows how the components are linked and organized to perform a task.

There are mainly two types of Architecture in Networking :



1. Peer to peer
2. Client server architecture





1. Peer-to-Peer :

In Peer to peer all the devices or components are connected to each other in different topologies to share data. All the computers are connected to every computer directly or indirectly. All the computers share the same resource and do not have any dedicated central computer(server).

Advantages:

• It is less costly
• All the computers manage their own task or work.
• In case of failure all the computers are not affected, all the computers are going to work as same as before the failure.


Disadvantages :

• Each and every computer must have to take backup because there is no server.
• Security issue.
• Headache in connecting to all the computers.


2. Client server architecture :
Client server architecture is a computer network in which a client can access the data or information like images songs videos etc. from a dedicated resource called server. In this all responsibilities are done by the server itself like security etc.
Advantages :
• Data Maintenance is easy because we don’t need to backup our data on every client machine.
• Security cause unauthorized client can not access tp the server.

Disadvantages :
• If server fails, the whole network will be affected.
• Server’s maintenance cost is too high


A Computer Network has lots of Networking components :



Networking components are those parts or devices which help us to establish a network. There are many components as hub, switch, router, cables, NIC, etc.

Lets take a brief knowledge about them :
• HUB is a hardware device which is used in a network to broadcast the request to the multiple clients on a network, from whom it belong that client takes it and other drop the request. It is used between two networks.

• Switch is very intelligent hardware device that connect multiple devices on a network. This device delivers the request to the correct destination using mac address of the destination. It also increases the speed of network.

• Router is als a hardware device which also used to connect lan type of network. It connects one network to another type of network. It sends the data through the best route by its own intelligence. It is a very common and necessary component of a network.

• NIC(network interface card) is a hardware device which is used for connecting one computer to another on a network to transfer data. This an be identified uniquely by the mac address hard coded onto it. They are of two types

• wired and wireless NIC is present inside the computer’s motherboard. Wireless NIC can be seen in a laptop whose connector catches the signal of a wireless network.


A Computer Network topology :



A network topology defines in which order or structure all the components are interconnected to each other.

There are many topologies as bus, star, ring, tree, mesh, hybrid
• In bus topology all the components are connected through a single cables to each other. Each node is connected to the cable directly. If a computer sends the message , that message will go through all the computers connected to that network. In this case if cables fails the whole network will be failed.

• In Ring topology every computer is connected to two other computers , in this data flow in one direction, if any computer crashes automatically all the computers will be down.

• Star topology is like star because in this all the computers are connected to one computer or server and all the data is sent through this server. In this if central device gets failed the entire network will be failed. This is the most used type topology .

• In Tree topology all the computers are connected to each others hierarchically . the top node is called root and all the child nodes are called decendents. It can be easily expended and wiring is point to point.
