Q.What is Networking?


In simple words networking is nothing but the combination or connection of two or more than two devices through which we can communicate with each and every device to share data or information. These devices are called nodes in networking. All the devices are interonnected to each other through the links.

A computer network has lots of components like :


1. Switch
2. Hub
3. NIC (network interface card)
4. Router
5. Modem
6. Cables and connectors etc.





A Computer Network has some Advantages like :


• Computer networking make our work very easy and fast.
• It is reliable.
• It provides flexibility.
• Secure.

A Computer Network has some Disadvantages like :


• Viruses and malwares
• Costly.
• Network crash problem
